Google Summer of Code -Selection

For the last 3 months, I have been working on a honeypot project after seeing the GSOC 2019 ideas page. So I chose the OWASP Honeypot project for GSOC as this project is mainly written in Python and I have been working on Python since 4 years now, so it was easy for me to read their codebase and contribute to them and send PR's.

About the Project

OWASP Honeypot is an open source software in Python language which designed for creating honeypot and honeynet in an easy and secure way!
My work is to test all the modules in this project and then when all the modules are tested and fixed by me ,I would continue to work on the WEB part of the project.The WEB part of the project now basically has a link to see some visualizations of the hacker's activity and the visualizations just use IP address and Country name. I would discuss with my mentor and extract some more knowledge to detect hacker activity easily which would make the Honeypot system more efficient.
